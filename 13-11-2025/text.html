Text Formatting

Learn how to format text in HTML.

What is Text Formatting in HTML?

Text formatting tags in HTML help to style, highlight, or structure text on a webpage.

Why do we use Text Formatting?

  • To highlight important text.
  • To make content readable.
  • To apply bold, italic, underline, etc.
  • To show quotes, code, and special text.

Common Text Formatting Tags:

  1. <b> – Makes text bold.
  2. <strong> – Shows important text (bold).
  3. <i> – Makes text italic.
  4. <em> – Emphasizes text (italic).
  5. <u> – Underlines text.
  6. <mark> – Highlights text.
  7. <small> – Makes text smaller.
  8. <del> – Shows deleted (cut) text.
  9. <ins> – Shows inserted (underlined) text.
  10. <sub> – Subscript text (H2O).
  11. <sup> – Superscript text (X2).

Other Useful Formatting Tags:

  1. <blockquote> – Long quotations.
  2. <q> – Short inline quote.
  3. <code> – Shows code snippets.
  4. <pre> – Displays preformatted text.

-: Text Formatting Examples :-

1. Bold Text:

<b>This is bold</b>

2. Italic Text:

<i>This is italic</i>

3. Underlined Text:

<u>This is underlined</u>

4. Highlighted Text:

<mark>This is highlighted</mark>

5. Deleted Text:

<del>Old Price: ₹999</del>

6. Inserted Text:

<ins>New Price: ₹799</ins>

7. Subscript:

H<sub>2</sub>O

8. Superscript:

X<sup>2</sup>

9. Code Example:

<code>print("Hello World")</code>

10. Blockquote:

<blockquote>This is a long quotation.</blockquote>
