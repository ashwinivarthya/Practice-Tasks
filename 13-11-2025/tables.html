HTML Tables

Learn how to create and structure tables in HTML

What is a table in a HTML?

A table in HTML is used to display information in rows and columns.

Why do we use Tables?

  • To Show data Clearly.
  • To compare Information.
  • To display Schedules.
  • To make lists easier to read.

Basic table structure & Table Tags:

An HTML table is made using the <table> tag.

  1. <table>: Creates the table.
  2. <tr>: Creates a row in the table.
  3. <th>: Creates a header cell in the table(bold by default).
  4. <td>: Creates a normal data cell in the table.

Table Features:

  1. Rowspan: Used to merge two or more rows.
  2. Colspan: Used to merge two or more columns.
  3. Caption: Adds a title to the table.
