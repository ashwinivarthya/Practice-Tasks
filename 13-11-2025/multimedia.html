Audios & Videos

Learn how to embed audio and video content in HTML.

<audio> Element :-

The HTML <audio> element is used to play an audio files on a web page.

To play an audio file in HTML, use the <audio> element as shown below:

<audio controls> <source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> Your browser does not support the audio element. </audio>

How it works is shown below:

The controls attribute adds audio controls, like play, pause, and volume.

The <source> element allows you to specify alternative audio files which the browser may choose from. The browser will use the first recognized format.

The text between the <audio> and </audio> tags will only be displayed in browsers that do not support the <audio> element.

To start an audio file automatically, use the autoplay attribute.

To mute an audio file, use the muted attribute.

<video> Element :-

The HTML <video> element is used to show a video on a web page.

To show a video in HTML, use the <video> element as shown below:

<video controls> <source src="video.mp4" type="video/mp4"> <source src="video.ogg" type="video/ogg"> Your browser does not support the audio element. </video>

How it works is shown below:

The controls attribute adds video controls, like play, pause and volume.

It is a good idea to always include width and height attributes. If height and width are not set, the page might flicker while the video loads.

The <source> element allows you to specify alternative video files which the browser may choose from. The browser will use the first recognized format.

The text between the <video> and </video> tags will only be displayed in browsers that do not support the <video> element.

To start a video automatically, use the autoplay attribute.

To mute a video, use the muted attribute.
