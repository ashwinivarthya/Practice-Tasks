CSS Syntax

Understandng CSS Selectors, Properties and Values.

What is CSS Syntax?

CSS syntax is the rule structure that tells the browser how to style HTML elements. Every CSS rule has a selector, property, and value.

Basic CSS Syntax

selector {
    property: value;
}

Example:

h1 {
    color: red;
    font-size: 30px;
}

This means: “Select all <h1> elements and apply red color and 30px font size.”

Parts of CSS Syntax

  • Selector: Selects the HTML element (e.g., p, h1, div)
  • Property: The style you want to apply (e.g., color, margin, width)
  • Value: The setting for the property (e.g., blue, 20px, center)

CSS Declaration Block

The part inside curly braces { } is called the declaration block.

div {
    background-color: lightgray;
    padding: 10px;
}

Multiple Declarations

You can write multiple properties inside one selector:

p {
    color: green;
    text-align: center;
    font-weight: bold;
}

CSS Syntax Rules

  • End each property with a semicolon (;)
  • Use curly braces ({ }) to group styles
  • Write property and value with a colon (:)
