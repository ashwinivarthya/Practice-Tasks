CSS Selectors

Learn how to target HTML elements with CSS.

What Are CSS Selectors?

CSS selectors are used to select the HTML elements you want to style. They tell CSS: “Which element should receive this style?”

1. Element Selector

Selects all HTML elements of the same type.

p {
    color: blue;
}

2. Class Selector

Used to select elements with a class attribute. A class always starts with a dot ( . ).

.example {
    background-color: yellow;
}

3. ID Selector

Selects a single unique element using its id. An id always starts with #.

#heading {
    font-size: 30px;
}
