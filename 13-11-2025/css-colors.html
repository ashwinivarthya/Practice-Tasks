CSS Colors

Working with colors in CSS.

What Are CSS Colors?

CSS allows you to apply colors to text, backgrounds, borders, and many other elements. Colors can be defined in multiple ways such as names, RGB, HEX, HSL, etc.

1. Color Names

Simple predefined color names.

p {
    color: red;
}

2. RGB Colors

Colors are created using Red, Green, and Blue values. Each value ranges from 0 to 255.

h1 {
    color: rgb(255, 0, 0);   /* Red */
}

3. RGBA Colors

Same as RGB, but includes Alpha (transparency) value (0 to 1).

div {
    background-color: rgba(0, 0, 255, 0.5);  /* Transparent Blue */
}

4. HEX Colors

Uses a six-digit hexadecimal code.

p {
    color: #00ff00;  /* Green */
}

5. HSL Colors

HSL stands for Hue, Saturation, Lightness.

p {
    color: hsl(200, 100%, 50%);
}

6. HSLA Colors

HSL with an Alpha transparency value.

div {
    background-color: hsla(300, 80%, 60%, 0.4);
}

7. Applying Background Colors

body {
    background-color: lightgray;
}

8. Applying Border Colors

div {
    border: 2px solid rgb(0, 128, 0);
}

9. Gradient Colors (Linear Gradient)

Used to create smooth color transitions.

.box {
    background: linear-gradient(red, yellow);
}

10. Gradient Colors (Radial Gradient)

.circle {
    background: radial-gradient(circle, blue, white);
}
