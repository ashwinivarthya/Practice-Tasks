CSS Margins & Padding

Controlling space Outside & Inside elements.

1. What is Margin?

Margin is the space outside an element. It creates space between two elements.

div {
    margin: 20px;
}

Margin – Individual Sides:

div {
    margin-top: 10px;
    margin-right: 15px;
    margin-bottom: 10px;
    margin-left: 15px;
}

Margin Shorthand:

Shorthand uses 1, 2, 3, or 4 values.

div {
    margin: 10px 20px 15px 5px;  
}

Margin Auto (Centering):

Used to center block elements.

div {
    width: 300px;
    margin: auto;
}

2. What is Padding?

Padding is the space inside an element. It adds space between content and border.

div {
    padding: 20px;
}

Padding – Individual Sides:

div {
    padding-top: 15px;
    padding-right: 10px;
    padding-bottom: 15px;
    padding-left: 10px;
}

Padding Shorthand:

div {
    padding: 10px 20px;
}

3. Margin vs Padding

  • Margin = Outside space
  • Padding = Inside space
  • Padding increases element size; margin does not.

4. Example Box:

.box {
    margin: 20px;
    padding: 15px;
    border: 2px solid black;
}
