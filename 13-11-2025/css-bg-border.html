CSS Backgrounds & Borders

Setting backgrounds for elements & Styling element borders.

1. Background Color

Used to set the background color of any element.

div {
    background-color: lightblue;
}

2. Background Image

Sets an image as the background.

body {
    background-image: url("bg.jpg");
}

3. Background Repeat

Controls whether the background image repeats.

div {
    background-repeat: no-repeat;
}

4. Background Size

Used to resize the background image.

div {
    background-size: cover;
}

5. Background Position

Sets image position inside the background area.

div {
    background-position: center;
}

6. Background Attachment

Makes the background image fixed when scrolling.

div {
    background-attachment: fixed;
}

7. Border Style

Defines the style of the border.

p {
    border-style: solid;   /* solid, dotted, dashed, double */
}

8. Border Width

p {
    border-width: 3px;
}

9. Border Color

p {
    border-color: red;
}

10. Border Shorthand

p {
    border: 2px solid blue;
}

11. Rounded Borders

Use border-radius to create rounded corners.

div {
    border: 2px solid black;
    border-radius: 15px;
}

12. Circle Shape Using Border-Radius

img {
    width: 150px;
    height: 150px;
    border-radius: 50%;
}
