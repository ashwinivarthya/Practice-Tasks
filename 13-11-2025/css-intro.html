Introduction to CSS

Learn the basics of Cascading Style Sheets.

What is CSS?

CSS (Cascading Style Sheets) is used to style and design HTML pages. It controls colors, spacing, layouts, fonts, borders, backgrounds, and more.

Why Do We Use CSS?

  • To make web pages look attractive
  • To separate design from structure
  • To control the layout of multiple pages at once
  • To improve user experience with clean design

Types of CSS

  • Inline CSS: Written inside the HTML tag using the style attribute.
  • Internal CSS: Written inside the <style> tag in the HTML file.
  • External CSS: Written in a separate .css file and linked using <link>.

What Can CSS Control?

  • Text color, size, and fonts
  • Background colors and images
  • Spacing (margin, padding)
  • Borders and shapes
  • Page layout (flexbox, grid)

Simple Example


p {
    color: blue;
    font-size: 18px;
}

This CSS will make all paragraph text blue and slightly larger.
