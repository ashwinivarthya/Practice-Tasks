Attributes

Learn how to use attributes in HTML elements.

What are HTML Attributes?

Attributes are special values added to HTML tags to give extra information or control how an element behaves.

Why do we use Attributes?

  • To add additional information to elements.
  • To control appearance and behavior.
  • To identify elements uniquely.
  • To link images, pages, or scripts.

Common HTML Attributes:

  1. id – Gives a unique name to an element.
  2. class – Groups elements for styling.
  3. style – Adds inline CSS.
  4. href – Used in links to specify the URL.
  5. src – Specifies the image or media source.
  6. alt – Provides alternate text for images.
  7. title – Shows tooltip text on hover.
  8. type – Used inside form inputs (text, email, password).

Extra Useful Attributes:

  1. width / height – Controls element size.
  2. placeholder – Shows hint text in input fields.
  3. value – Sets the default value in a form.
  4. disabled – Makes a button or input non-clickable.
  5. readonly – Allows viewing but not editing.
  6. required – Makes input mandatory.

-: Attribute Examples :-

1. Image with src and alt:

<img src="photo.jpg" alt="My Photo">

2. Link with href:

<a href="https://google.com">Visit Google</a>

3. Input with placeholder:

<input type="text" placeholder="Enter your name">

4. Button with id and class:

<button id="submitBtn" class="btn">Submit</button>

5. Paragraph with inline style:

<p style="color: blue;">This is styled text.</p>
