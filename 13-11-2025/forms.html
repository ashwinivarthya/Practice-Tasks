HTML Forms

Learn how to create interactive forms in HTML

What is a Form in HTML?

An HTML form is used to collect information from the user.

Why do we use Forms?

  • To collect user data.
  • For registrations and Logins.
  • For feedback and surveys.
  • For searching.
  • For contact information.
  • For booking tickets or filling applications.

Basic Form Structure & Tags:

The main form is created using the <form> tag.

  1. <form>: Creates a form.
  2. <label>: Adds a label for inputs.
  3. <input>: Takes user input.
  4. <textarea>: For long text input.
  5. <select>: Dropdown list.
  6. <option>: Options inside dropdown.
  7. <button>: Creates buttons(submit/reset).

Common Input Types:

  1. text: normal text.
  2. email: email input.
  3. password: hide text.
  4. number: numbers only.
  5. radio: select one option.
  6. checkbox: select multiple options.
  7. date: pick a date.
  8. file: Upload a file.
  9. submit: submit the form.
