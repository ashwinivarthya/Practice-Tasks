-:HTML Tables:-

Tables in HTML are used to display data in a tabular format. They are created using the <table> element, with rows defined by <tr> and cells by <td> (for data cells) or <th> (for header cells).

A table cell can contain all sorts of HTML elements: text, images, lists, links, other tables, etc.,

You can have as many rows as you like in a table; just make sure that the number of cells are the same in each row.

Basic Table Structure:

An HTML table is defined using the <table> tag.

Each table row is inside a <tr> tag (table row).

Each data cell is inside a <td> tag (table data).

Each heading cell is inside a <th> tag (table header).

View Example

Table Attributes:

You can use various attributes to style and format tables, such as:

  • border: Specifies the width of the table border.
  • cellpadding: Specifies the space between cell content and cell borders.
  • cellspacing: Specifies the space between cells.

Table Headers and Footers:

The <thead>, <tbody>, and <tfoot> elements are used to group header, body, and footer content in a table.
