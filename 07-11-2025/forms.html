-:HTML Forms:-

  1. An HTML form is used to collect user input. The user input is most often sent to a server for processing.
  2. The HTML <form> element is used to create an HTML form for user input.
  3. The <form> element is a container for various types of input elements such as text fields, checkboxes, radio buttons, and submit buttons.etc.,
  4. The HTML <input> element is the most used form element.
  5. An <input> element can be displayed in many ways, depending on the type attribute.
  6. The <label> tag defines a label for many form elements.
  7. The <label> element also helps users who have difficulty clicking on very small regions (such as radio buttons or checkboxes) - because when the user clicks the text within the <label> element, it toggles the radio button/checkbox.
  8. The for attribute of the <label> tag should be equal to the id attribute of the <input> element to bind them together.
  9. The <input type="radio"> defines a radio button. Radio buttons allow the user to select one option from a set.
  10. The <input type="checkbox"> defines a checkbox. Checkboxes allow the user to select one or more options from a set of choices.
  11. The <input type="submit"> defines a button for submitting the form data to a form-handler.

i. Basic Structure of an HTML Form

<form action="submit_form.php" method="post"> <label for="name">Name:</label> <input type="text" id="name" name="name"> <input type="submit" value="Submit"> </form>

Example of a Simple HTML Form:

<form action="/submit_form" method="post"> <label for="username">Username:</label> <input type="text" id="username" name="username"><br> <label for="password">Password:</label> <input type="password" id="password" name="password"><br> <input type="checkbox" id="remember" name="remember"> <label for="remember">Remember me</label><br> <input type="submit" value="Login"> </form>

ii. HTML Form Elements

  • <form>: Defines an HTML form for user input.
  • <input>: Used for various types of input fields (text, password, checkbox, radio, etc.).
  • <textarea>: Used for multi-line text input.
  • <select>: Used to create a drop-down list.
  • <button>: Used to create clickable buttons.
  • <label>: Defines labels for input elements.
  • <option>: Defines an option that can be selected.
  • <fieldset>: Groups related elements in a form.
  • <legend>: Defines a caption for a <fieldset> element.

iii. Form Attributes

  • action: Specifies the URL where the form data will be sent upon submission.
  • method: Specifies the HTTP method to use when sending form data (GET or POST).
  • enctype: Specifies how the form data should be encoded when submitted (used for file uploads).
  • target: attribute tells the browser where to open the response after form submission.

    • There are two targets we use very often those are _blank and _self.

    Where _blank is used to open the response in next window. Whereas _self opens in the same window.
